# smart alarm

Smart Alarm is a React Native app that calculates your wake-up time based on upcoming calendar events.

The app uses mock calendar data to find the earliest event of the day and sets an alarm a configurable amount of time before it.

## Tech Stack
- React Native
- Expo
- JavaScript

## Features
- Calendar-based alarm calculation (mock data)
- Adjustable buffer time before first event
- Simple, clean mobile UI
- Runs on iOS, Android, and web via Expo

## Status
Early prototype focused on UI and scheduling logic.
